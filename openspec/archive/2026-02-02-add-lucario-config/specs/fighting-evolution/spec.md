# Specifications: Fighting Evolution Line

## Requirement: Lucario Configuration
The system SHALL define the `lucario` character configuration.
- **HP**: 240
- **Speed**: 250
- **Base Damage**: 40
- **Passive**: `innerFocusPassive`
- **Weapon**: `auraSphere` (reused from Riolu)
- **Ultimate**: `boneRush` (reused from Riolu)
- **Archetype**: `archetype_aura_guardian`

## Requirement: Evolution Chain
The system SHALL link the evolution path:
- **Riolu** -> **Lucario** (Level 20)
- **Lucario** -> None (Final Form)
