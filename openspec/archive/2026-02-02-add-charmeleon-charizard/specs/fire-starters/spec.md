# Specifications: Fire Starters Evolution

## Requirement: Charmeleon Configuration
The system SHALL define the `charmeleon` character configuration.
- **HP**: 280
- **Speed**: 210
- **Base Damage**: 35 (1.8x multiplier equivalent)
- **Passive**: `blazePassive`
- **Weapon**: `ember`
- **Ultimate**: `seismicToss`
- **Archetype**: `archetype_pyro_bruiser`

## Requirement: Charizard Configuration
The system SHALL define the `charizard` character configuration.
- **HP**: 450
- **Speed**: 260
- **Base Damage**: 65 (3.0x multiplier equivalent)
- **Passive**: `blazePassive`
- **Weapon**: `ember`
- **Ultimate**: `seismicToss`
- **Archetype**: `archetype_mobile_artillery`

## Requirement: Evolution Chain
The system SHALL link the evolution path:
- **Charmander** -> **Charmeleon** (Level 16)
- **Charmeleon** -> **Charizard** (Level 36)

*Note: Levels 16/36 are standard Pokemon evolution levels. I will assume these for the spec unless user specified otherwise (they didn't specify levels, just configs).*
