# Redesign Stealth Rock Visuals

## Why
The user requested a redesign because the current "simple brown circle" does not look like "Stealth Rock". Reference images show jagged, floating, sharp rocks.

## What Changes
- **Visuals**: Replace circle sprite with a procedural jagged rock texture (Polygon).
- **Animation**: Add rotation to simulate floating/tumbling stones.
- **Variety**: Slight size/rotation variance per rock.

## Verification
- Visual inspection ensuring jagged shapes.
- Gameplay check ensuring hitbox behavior remains consistent (or improves).
