# map-save-ui Specification

## Purpose
The Map Save UI provides a consistent interface for managing map files, including list-based navigation and selection for save and load operations.
## Requirements
### Requirement: Map Gallery Display
The map save and load interfaces SHALL display a gallery or list of available maps.

#### Scenario: Responsive map list
- **WHEN** multiple maps are available
- **THEN** they SHALL be displayed in a scrollable list or grid within the modal
