# Specifications: Water Evolution Line

## Requirement: Wartortle Configuration
The system SHALL define the `wartortle` character configuration.
- **HP**: 320
- **Speed**: 160
- **Base Damage**: 30
- **Passive**: `rainDishPassive`
- **Weapon**: `waterPulse`
- **Ultimate**: `shellSmash`
- **Archetype**: `archetype_shell_guardian`

## Requirement: Blastoise Configuration
The system SHALL define the `blastoise` character configuration.
- **HP**: 550
- **Speed**: 190
- **Base Damage**: 55
- **Passive**: `rainDishPassive`
- **Weapon**: `waterPulse`
- **Ultimate**: `shellSmash`
- **Archetype**: `archetype_siege_fortress`

## Requirement: Evolution Chain
The system SHALL link the evolution path:
- **Squirtle** -> **Wartortle** (Level 20)
- **Wartortle** -> **Blastoise** (Level 40)
