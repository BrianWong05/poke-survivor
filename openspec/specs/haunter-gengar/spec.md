# Specifications: Ghost Evolution Line

## Requirement: Haunter Configuration
The system SHALL define the `haunter` character configuration.
- **HP**: 140
- **Speed**: 265
- **Base Damage**: 24
- **Passive**: `shadowTagPassive`
- **Weapon**: `lick`
- **Ultimate**: `destinyBond`
- **Archetype**: `archetype_shadow_stalker`

## Requirement: Gengar Configuration
The system SHALL define the `gengar` character configuration.
- **HP**: 250
- **Speed**: 310
- **Base Damage**: 45
- **Passive**: `shadowTagPassive`
- **Weapon**: `lick`
- **Ultimate**: `destinyBond`
- **Archetype**: `archetype_nightmare_weaver`

## Requirement: Evolution Chain
The system SHALL link the evolution path:
- **Gastly** -> **Haunter** (Level 25)
- **Haunter** -> **Gengar** (Level 40 or Trade - will use Level 40 for simplicity as per game mechanics)

*Note: Standard evolution for Gastly is Lv 25. Haunter evolves by trade, but in this Survivor-like game, we usually use levels. I will use Lv 40 for Gengar unless specified otherwise.*
